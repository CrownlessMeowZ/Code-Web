Vocaloid Characters In Project DIVA

Hatsune Miku, officially code-named CV01, is a Vocaloid software voicebank developed by Crypton Future Media.

Kagamine Rin, officially code-named CV02, is a Vocaloid software voicebank developed by Crypton Future Media.

Kagamine Len, officially code-named CV02, is a Vocaloid software voicebank developed by Crypton Future Media.

Megurine Luka, officially code-named CV03, is a Vocaloid software voicebank developed by Crypton Future Media.

Kaito is a Vocaloid software voicebank developed by Crypton Future Media, officially recognized as the first Japanese male voicebank released for the Vocaloid 1 engine.

Meiko is a Vocaloid software voicebank developed by Crypton Future Media, officially recognized as the first Japanese female voicebank released for the Vocaloid 1 engine.
